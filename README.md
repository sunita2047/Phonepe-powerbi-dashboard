# 📊 PhonePe Business Analytics Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using PhonePe transaction and user datasets. The dashboard provides insights into transaction performance, revenue trends, customer behavior, and service usage patterns.

The objective of this project is to transform raw transaction data into meaningful business insights through interactive visualizations and KPI reporting.

## 🎯 Business Objectives

The dashboard aims to answer key business questions such as:

* What is the total revenue generated?
* How many users and transactions are recorded?
* Which services generate the highest revenue?
* What are the transaction success and failure rates?
* Which users contribute the most to revenue?
* How does revenue vary across age groups and time periods?

## 📂 Dataset Information

The project uses two datasets:

### Users Dataset

Contains information related to users including:

* User ID
* User Name
* Age
* Join Date

### Transactions Dataset

Contains transaction-related information including:

* Transaction ID
* User ID
* Transaction Amount
* Service
* Payment Status
* Failure Reason
* Transaction Date
